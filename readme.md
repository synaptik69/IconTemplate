#### Install

```
npm install react-icomoon
```

Use +498 free icon.
Visit the IcoMoon site to see the icon list.

https://icomoon.io/app/

#### Usage

```js
import IcoMoon from "react-icomoon";
```

```js
<IcoMoon icon="pencil" />
```

```js
<IcoMoon icon={value ? "play" : "pause"} />
```
