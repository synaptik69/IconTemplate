## React IconMoon

#### Install
```
npm install react-iconmoon
```


#### Usage
```js
import IconMoon from 'react-iconmoon';

// ...

class Example extends React.Component {
    render(){
        return (
            <IconMoon icon="pencil" />
        )
    }
}
```

Visit the IconMoon site to see the icon list.

https://icomoon.io/app/
